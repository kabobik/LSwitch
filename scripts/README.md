# Скрипты LSwitch

Служебные скрипты для установки и диагностики:

## Установка
- `install.sh` - основной скрипт установки
- `uninstall.sh` - удаление программы
- `setup_service.sh` - настройка systemd сервиса

## Диагностика
- `diagnose.sh` - диагностика проблем
- `check_autostart.sh` - проверка автозапуска
- `test_quick.sh` - быстрое тестирование
