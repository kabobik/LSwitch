# Ассеты LSwitch

## Иконки

### Программно генерируемые (для трея)
Иконка в системном трее раньше создавалась программно в `lswitch_tray.py` — этот модуль был архивирован и перемещён в `archive/removed_tray`.
- **Белая** иконка для тёмной темы
- **Тёмная** иконка для светлой темы

### PNG иконки
Сгенерированные PNG иконки в разных размерах:

**Для тёмной темы (белые):**
- `lswitch-light-{16,22,24,32,48,64,128,256}.png`

**Для светлой темы (тёмные):**
- `lswitch-dark-{16,22,24,32,48,64,128,256}.png`

### Генерация иконок

Для повторной генерации всех размеров:

```bash
python3 generate_icons.py
```

Это создаст 16 PNG файлов (8 размеров × 2 варианта цвета).

## Старые файлы

- `lswitch.svg` - оригинальная SVG иконка
- `create_icon.py` - старый скрипт создания иконки

## Дизайн иконки

Упрощённая иконка клавиатуры:
- Основной корпус с rounded corners
- 2 ряда по 6 клавиш
- Длинный пробел внизу
- Эффект углублений (вырезанные клавиши)
