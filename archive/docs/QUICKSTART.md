# LSwitch — Быстрый старт

## Установка

```bash
bash scripts/install.sh
```

Выберите "y" для включения автозапуска.

## Команды

```bash
make start       # Запустить
make stop        # Остановить
make restart     # Перезапустить
make status      # Статус
make logs        # Логи

make enable      # Автозапуск ON
make disable     # Автозапуск OFF

make uninstall   # Удалить
```

## Использование

1. Печатайте текст в любом приложении
2. Набрали не в той раскладке?
3. **Быстро нажмите Shift дважды**
4. Последнее слово конвертируется!

## Примеры

- `ghbdtn` → двойной Shift → `привет`
- `руддщ` → двойной Shift → `hello`

## GUI

```bash
lswitch-control
```

Открывает панель управления для настройки параметров.
